# High-Performance E-Commerce Engine with AI Vector Search

This project is a modern, high-performance e-commerce engine built with Node.js, Express, React, Vite, MongoDB, and Redis. It features local AI Vector Search utilizing Hugging Face Transformers for intelligent product discovery.

## Features
- AI-Powered Vector Search
- Lightning-fast Redis Caching
- Responsive React Frontend with Vite
- Express Backend with MongoDB
